#deep_clone 深度克隆
#####JavaScript对象深度克隆思路：
```
    1. 创建一个函数，判断所传入对象的类型，主要区别Object类型和Array类型
        1.1 Object.prototype.toString.call(对象) 这段代码可用来判断对象类型，返回的结果是'[object 类型]' 
        1.2 可通过splice(8, -1)截取字符串，得到类型
    2. 创建一个深度克隆方法
        2.1 定义一个空对象用来保存克隆结果
        2.2 定义一个变量用来保存对象类型
        2.3 通过判断所传入对象的类型，来确定用来保存克隆结果的空对象的类型
        2.4 通过循环，进行深复制，对象属性仍是对象则调用递归方法进一步取值赋值
        2.5 返回结果

```
#####深度克隆原理：
```
    原始类型以及对象类型中的函数是可以直接通过“=”复制来实现克隆，而对于对象和数组，则需要遍历每一个元素，如果元素为对象或数组，通过“=”复制，得到的只是一个堆地址，无法达到深复制，为了达到深复制则需继续遍历，直到为原始类型或函数，再通过“=”进行复制。
```
